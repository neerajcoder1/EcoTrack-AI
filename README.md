# EcoTrack AI

## Measure. Understand. Reduce.

EcoTrack AI is an AI-powered Carbon Footprint Awareness Platform designed to help individuals understand their environmental impact and adopt sustainable habits through data-driven insights and personalized recommendations.

## Problem Statement

Many people are unaware of how their daily activities contribute to carbon emissions. EcoTrack AI helps users calculate, monitor, and reduce their carbon footprint through an intuitive dashboard, AI-generated recommendations, and sustainability tracking tools.

## Installation

```bash
git clone https://github.com/your-username/EcoTrack-AI.git

cd EcoTrack-AI

npm install

npm run dev
```

## Environment Variables

Create a .env file and add:

```env

NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
```
